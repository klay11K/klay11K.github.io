Hexo 部署到 github

  1. 安装 hexo-deployer-git.

  2. 在 github 上新建项目，注意项目名一定要是 username.github.io，这个后续是你 Hexo 访问的域名！

  3. 配置 Hexo 根目录下的 _config.yml 文件。

  4. 执行命令

  5. 访问域名：username.github.io，我的域名是：klay11K.github.io，可以访问。
